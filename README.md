# MorseTranslator
App that can be used for translating Alphabet-Morse or Morse-Alphabet

### Used Library : 
- TextScanner (https://github.com/LucemAnb/Android-Text-Scanner)
- MaterialDesign (https://material.io)
- Room (https://developer.android.com/jetpack/androidx/releases/room)
